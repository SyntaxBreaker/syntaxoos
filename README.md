# Syntaxoos
**SyntaxoOS** is a simple, web-based operating system simulator. Built with a modern frontend stack, it changes a browser tab into a desktop environment.

---
## Desktop Experience
These features make the website feel like a real computer:
* **Boot Screen:** A basic loading screen that appears when you "turn on" the OS.
* **Panel:** A taskbar to see what windows are running.
* **Context Menu:** Right-click anywhere to see a custom menu, just like a real PC.
* **Settings:** Change your account info or pick a new look.
---
## Built-in Apps
* **File Manager:** Open folders and look through saved files.
* **Task Manager (ToDo):** A simple list to help you finish your daily chores.
* **System Monitor:** A window that shows how well the system is running.
* **Terminal:** A text-based tool for people who like to type commands.
---
## Creative & Media Tools
* **Music Player:** Listen to music.
* **Image Viewer:** A clean app to look at photos.

## Useful Utilities
* **Calculator:** This handles everything from 2+2 to more advanced math problems.
* **Dictionary:** Quickly look up words you don't know.
* **Contact:** An easy way to find contact with the creator of the project.
* **Portfolio:** A special app that showcases my projects in one place.
